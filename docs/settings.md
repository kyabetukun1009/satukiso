- 設置ページ：/contact/
- 入力項目：
  - メールアドレス（必須）
  - 電話番号（任意）
  - お問い合わせ種別（必須：宿泊・ご予約 / 五月荘について / 広告掲載・取材 / その他）
  - お問い合わせ内容（必須）
  - プライバシーポリシー同意（必須）
- 通知メール設定：
  - 送信先：satukiso@proton.me
  - 送信元：五月荘 <noreply@kyabetukun.dpdns.org>
  - Reply-To：ユーザー入力メール（[your-email]）
  - 件名：お問い合わせフォーム経由（種別を件名に反映）
- メール形式：
  - HTMLメール：有効
  - 空行抑制：有効
- 送信方式：
  - InfinityFree 環境では PHP mail() が制限されるため、SMTP（WP Mail SMTP / Gmail API）を使用しているが有料のサーバーであれば本来は不要になるはず

## SEO設定（Rank Math SEO）

- 使用ツール：Rank Math SEO（WordPressプラグイン）
- 基本方針：
  - サイト構築・検証段階のため、現時点では検索結果への
    インデックス登録を行わない方針
- 固定ページ設定：
  - トップページ、部屋紹介、利用案内、アクセス、
    お問い合わせページに title および meta description を個別設定
  - SNS表示用のOGPタグを各ページに設定
- インデックス設定：
  - 通常固定ページを含め、全ページを noindex に設定
  - 本番運用開始時に index 設定へ切り替え予定
- 404ページ：
  - 専用の title を設定
  - noindex を指定
- 未対応項目（今後の改善候補）：
  - XMLサイトマップ
  - robots.txt のカスタマイズ
  - Google Search Console / Analytics 連携
  - サイトファイル及び画像の圧縮
  - 構造化データなどなど

## DNS / SSL 設定（Cloudflare）

- DNS 管理：Cloudflare を使用
- 接続方式：
  - A レコードおよび CNAME レコードを利用し、
    本番サーバー（InfinityFree）へ接続
- SSL：
  - Cloudflare による HTTPS を有効化
- キャッシュ設定：
  - キャッシュ有効期限を 8 日に設定
- 備考：
  - その他の設定はデフォルトを使用

## 予約システム設定（Reserva）

- 使用サービス：Reserva（外部予約システム）
- 予約ページURL：
  - https://reserva.be/satukiso

### 基本設定
- 施設情報：
  - 住所、アクセス方法、連絡先情報を登録
  - 公式サイトへのリンクを設定
- 予約ページ：
  - 施設名・説明文・画像を設定
  - チェックイン／チェックアウト時刻を設定
  - 予約受付期限、キャンセルポリシーを設定
- 画像：
  - Webサイトで使用している画像素材を流用

### 客室タイプ設定
- 客室情報：
  - 客室名、説明文、注意事項を設定
  - 定員、部屋種別（和室／洋室）、禁煙設定を登録
  - 設備・アメニティ（Wi-Fi、空気清浄機 等）を登録
- 宿泊条件：
  - 宿泊可能日数を設定

### 料金設定
- 料金方式：
  - 一人あたり料金を採用
- 子供区分：
  - 0～5歳：無料（定員には含むが料金計算対象外）
  - 6～11歳：子供料金設定
  - 12～18歳：大人に準じた料金設定
- 祝日・祝前日：
  - 個別に料金設定
- 消費税：
  - システムにより自動計算
- 追加料金：
  - なし
- 長期滞在割引：
  - 未設定
